# 料金一覧
## トレース料金
### トレースの記録
- トレース 1 件あたり ```0.000005 USD```
- トレースをサンプリングして、X-Rayに保存する際の料金
- サンプリング割合を変更する事で料金を安くできる
### トレースの取得 
- トレース 1 件あたり ```0.0000005  USD```
- たぶん、クエリした際のスキャンをかけた後、実際に取得できたトレース数
### トレースのスキャン
- トレース 1 件あたり ```0.0000005  USD```
- たぶん、クエリした際のスキャンをかけたトレース数
