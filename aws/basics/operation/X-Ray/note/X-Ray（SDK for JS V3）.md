# 概要
PutTraceSegmentsCommand コマンドを使用して、X-Rayトレースを取得する。  
このSDKは[**X-Ray SDK**](https://docs.aws.amazon.com/xray-sdk-for-nodejs/latest/reference/)とは別物であり、X-Rayデーモンを建てる必要もない。
