# 概要
コンソール上で「デプロイグループの作成」を実施する際の各種項目を解説する。

# 各種設定項目解説
## デプロイ設定
### 元のリビジョンの終了
- Blue/Greenデプロイが完了し、切替前に使用していたECSのタスクを何時間後に終了させるか設定できる。
- もし、新しいタスクに問題があった場合にすぐにロールバックできるよう、切替前タスクの停止時間を設定する。
